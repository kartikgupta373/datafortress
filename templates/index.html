Frequently Asked Questions

Some commonly asked questions and issues

  • How to use the application?

    The application requires the user to signup as a creator or an editor and share Files as per need.

  • How is the shared data secured?

    A Machine learning anomaly based Intrusion Detection System is being used which keeps the database and the files secure from cyberattacks.
